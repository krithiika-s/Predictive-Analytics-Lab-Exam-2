
# Predictive Analytics Lab Exam-2

## Objective

To perform a classification task using machine learning.

## Dataset

The dataset is stored in the `dataset` folder.

## Steps Performed

### 1. Exploratory Data Analysis (EDA)

* Used head(), info(), describe() to understand the dataset
* Checked structure, data types, and summary statistics

### 2. Model Building

* Used Logistic Regression for classification
* Split data into training and testing sets
* Applied feature scaling

### 3. Model Evaluation

* Evaluated using Confusion Matrix
* Calculated Accuracy score

## Results

* Accuracy: 0.5550   

## Conclusion

The model successfully classifies the data with good performance.
